# ExampleKotlinVelocityPlugin
An example plugin made in Kotlin for Velocity proxy servers.

## Building
> Requires installing Git

### Windows
```
git clone https://github.com/Miclebrick/ExampleKotlinVelocityPlugin.git
./gradlew.bat build
```

### Linux
```
git clone https://github.com/Miclebrick/ExampleKotlinVelocityPlugin.git
./gradlew build
```
